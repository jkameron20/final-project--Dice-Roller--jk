Dice Roller

Brogrammers

Last Updated: Sprint 5, 11/8/20

Application Uses

This application will allow the user to calculate the odds when rolling a die. The user will be able to roll a die thousands of times if neccessary to gather data.

Guaranteed Features

  1. Give an output when a six sided die is rolled
  2. Keep track of past rolls

Future Features

  1. Roll up to a 20 sided die
  2. Be able to rig a die and see the odds

Neccessary Languages and Platforms

  1. This application will be performed through python

Technical Features Learned in Class

  1. Fundamental use of python

Technical Features Learned Beyond Class

  1. The ability to program multiple functions in one application
